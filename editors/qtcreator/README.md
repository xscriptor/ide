# Qt Creator Xscriptor

Color themes for the [Qt Creator](https://www.qt.io/product/development-tools) IDE, generated from the Xscriptor palette collection.

## Installation

```bash
./install.sh
```

After installation, restart Qt Creator and select the theme from **Preferences > Environment > Interface > Theme**.

## Uninstall

```bash
./uninstall.sh
```
