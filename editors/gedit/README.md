## Installation

### Linux

```bash
cp themes/*.xml ~/.local/share/gedit/styles/
```

Or run the install script:

```bash
./install.sh
```

### macOS

```bash
cp themes/*.xml ~/Library/Preferences/org.gnome.gedit/styles/
```

## Usage

Select the theme in Gedit via **Preferences > Font & Colors**.

## Uninstall

```bash
./uninstall.sh
```
