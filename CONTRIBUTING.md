# Contributing

Thanks for your interest in contributing to this monorepo.

## How to contribute

1. Fork the repository.
2. Create a branch with a descriptive name.
3. Make your changes.
4. Submit a pull request describing what you changed and why.

## Guidelines

- Keep changes scoped to the relevant editor directory.
- Follow the existing style and conventions of the project you are modifying.
- If adding a new editor directory, include a README and a clear license.
- Document any new color palettes or theme formats.

## Code of conduct

This project follows a standard code of conduct. By participating, you agree
to maintain a respectful and inclusive environment for everyone.
