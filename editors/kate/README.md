## Installation

### Linux

```bash
cp themes/*.theme ~/.local/share/kate/color-schemes/
```

Or run the install script:

```bash
./install.sh
```

### macOS

```bash
cp themes/*.theme ~/Library/Application Support/Kate/color-schemes/
```

## Usage

Select the theme in Kate via **Settings > Color Themes**.

## Uninstall

```bash
./uninstall.sh
```
