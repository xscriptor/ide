# Xscriptor Themes for Nova

12 color themes for Nova (by Panic), generated from [colors.json](../../colors.json).

## Installation

```bash
./install.sh
```

Or manually copy the `.nvatheme` files from `themes/` to `~/Library/Application Support/Nova/Themes/`.

## Usage

1. Open Nova
2. Go to **Nova > Preferences > General**
3. Select a theme from the **Theme** dropdown

Available themes: X, Madrid, Lahabana, Miami, Paris, Tokio, Oslo, Helsinki, Berlin, London, Praha, Bogota.

## Uninstall

```bash
./uninstall.sh
```

## Regeneration

```bash
python3 generate.py
```
