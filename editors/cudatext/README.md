# Xscriptor Themes for CudaText

## Install

```bash
./install.sh
```

Or manually copy `.json` files to your CudaText `color` directory.

Common paths:

- **Linux**: `~/.config/cudatext/data/color/` or `py/cudatext/app/data/color/`
- **macOS**: `~/Library/Application Support/CudaText/data/color/`
- **Windows**: `%APPDATA%\CudaText\data\color\`

## Uninstall

```bash
./uninstall.sh
```

## Usage

In CudaText, go to **Options → Color Theme** and select an Xscriptor theme.
