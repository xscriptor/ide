# Xscriptor Themes for Replit

12 color themes for Replit, generated from [colors.json](../../colors.json).

## Installation

Replit themes are uploaded via the Replit UI:

1. Open Replit
2. Go to your **Account Settings** (profile icon > **Settings**)
3. Under **Themes**, click **Import Theme**
4. Upload the desired `.json` file from `themes/`

Available themes: X, Madrid, Lahabana, Miami, Paris, Tokio, Oslo, Helsinki, Berlin, London, Praha, Bogota.

## Uninstall

Replit themes can be removed from the **Themes** section in your account settings.

## Regeneration

```bash
python3 generate.py
```
