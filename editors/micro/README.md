# Xscriptor Themes for Micro

## Install

```bash
./install.sh
```

Or manually copy themes to `~/.config/micro/colorschemes/`.

## Uninstall

```bash
./uninstall.sh
```

## Usage

Open Micro and press `Ctrl+E`, then:

```
set colorscheme xscriptor-<name>
```

Replace `<name>` with one of: x, madrid, lahabana, miami, paris, tokio, oslo, helsinki, berlin, london, praha, bogota.
