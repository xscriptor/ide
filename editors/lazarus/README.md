# Lazarus IDE Xscriptor

Color themes for the [Lazarus IDE](https://www.lazarus-ide.org/), generated from the Xscriptor palette collection.

## Installation

```bash
./install.sh
```

After installation, select the theme in Lazarus via **Tools > Options > Editor > Colors**.

## Uninstall

```bash
./uninstall.sh
```
