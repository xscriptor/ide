# Xscriptor Themes for CodeSandbox

12 color themes for CodeSandbox, generated from [colors.json](../../colors.json).

## Installation

CodeSandbox themes are uploaded via the CodeSandbox UI:

1. Open CodeSandbox
2. Go to your **Settings** (profile icon > **Settings**)
3. Under **Appearance**, click **Import Theme**
4. Upload the desired `.json` file from `themes/`

Available themes: X, Madrid, Lahabana, Miami, Paris, Tokio, Oslo, Helsinki, Berlin, London, Praha, Bogota.

## Uninstall

CodeSandbox themes can be removed from the **Appearance** section in your settings.

## Regeneration

```bash
python3 generate.py
```
