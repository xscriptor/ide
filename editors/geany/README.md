## Installation

### Linux

```bash
cp themes/*.scheme ~/.config/geany/colorschemes/
```

Or run the install script:

```bash
./install.sh
```

## Usage

Select the theme in Geany via **View > Change Color Scheme**.

## Uninstall

```bash
./uninstall.sh
```
