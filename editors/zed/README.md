# Xscriptor Themes for Zed

12 color palettes (X, Madrid, Lahabana, Miami, Paris, Tokio, Oslo, Helsinki, Berlin, London, Praha, Bogota).

## Install

```bash
./install.sh
```

Or manually copy the `.json` files from `dist/` to `~/.config/zed/themes/`.

## Uninstall

```bash
./uninstall.sh
```

## Generate

```bash
python3 generate.py
```
