# Arduino IDE 2.x Xscriptor

Color themes for the [Arduino IDE 2.x](https://www.arduino.cc/en/software), generated from the Xscriptor palette collection.

## Installation

```bash
./install.sh
```

After installation, select the theme in Arduino IDE via **File > Preferences > Theme**.

## Uninstall

```bash
./uninstall.sh
```
